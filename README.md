# AqilliMed (React + TypeScript + Tailwind)

Bu loyiha kasalxona uchun zamonaviy frontend platforma prototipi:

- Hamshiralar uchun kompyuterda bemor registratsiyasi
- Bemorlar uchun online qabul band qilish va to‘lov checki yuborish oqimi
- Admin panelda doktor/bemor nazorati
- Doktorlarga yozilish statistikasi (diagramma)
- Bemor profilida qolish kunlari va jami to‘lov ko‘rinishi

## Ishga tushirish

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Eslatma

Hozirgi versiya frontend prototip. Real tizimga chiqarish uchun backend, autentifikatsiya,
haqiqiy online to‘lov integratsiyasi va Telegram Bot API ulanishini qo‘shish kerak bo‘ladi.
